A linked list usually only contains 2 attributes, it will have the head node, and the length of the current linked list
  • Head - The very first Node in the linked list
  • Length - Keeps track of how many nodes are in the Linked List, usually starts at 1
      
      function LinkedList () {
        this.head = null;
        this.length = 0; // Keeps track of length
      }
Each node in a singly linked list only contains two properties,
  • Value - The Node's value
  • Next - The Node that comes after it
        
      function Node (value, next) {
        this.value = value;
        this.next = next;
      }
now for example if we were to create a new linked list from our constructor 
        
      var ll = new LinkedList();
      console.log(ll);
we would get: 
        
      LinkedList { head: null, length: 0 }
